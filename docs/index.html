Baileys

WhatsApp Web API — Multi-device WhatsApp automation library

GitHub  |  Releases

Installation

npm install github:Manuel5906/baileys

Quick Start

const { default: makeWASocket, useMultiFileAuthState, DisconnectReason } = require('baileys');

async function start() {
    const { state, saveCreds } = await useMultiFileAuthState('auth_info');
    const sock = makeWASocket({
        auth: state,
        printQRInTerminal: true,
    });

    sock.ev.on('messages.upsert', async ({ messages }) => {
        const m = messages[0];
        if (!m.key.fromMe) {
            await sock.sendMessage(m.key.remoteJid, {
                text: 'Hello from Baileys!'
            }, { quoted: m });
        }
    });

    sock.ev.on('connection.update', ({ connection, lastDisconnect }) => {
        if (connection === 'close') {
            start();
        }
    });
}

start();

Features

Multi-Device

Full WhatsApp multi-device support via WebSocket.

Interactive Messages

Buttons, lists, native flow, copy-to-clipboard, and URL buttons.

Media Support

Images, videos, documents, audio messages with thumbnails.

Groups & Chats

Manage groups, polls, events, albums, and stories.

Session Management

Multi-file auth state, in-memory store, cache manager support.

Stable Pairing

Robust reconnection with customizable pairing code.

API Overview

  • makeWASocket(config) — Create a WhatsApp connection
  • sendMessage(jid, content, options) — Send any message type
  • useMultiFileAuthState(folder) — Persistent auth state
  • generateWAMessageFromContent(jid, content, options) — Generate messages
  • prepareWAMessageMedia(message, options) — Prepare media uploads
  • makeInMemoryStore() — In-memory message store

Button Types

  • cta_copy — Copy to clipboard button
  • cta_url — Open URL button
  • single_select — Single-select list menu
  • call_permission_request — Request call permission
  • review_and_pay — Review and pay
  • review_order — Review order
  • payment_info — Payment info
  • payment_status — Payment status
  • payment_method — Payment method
