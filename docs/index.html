Baileys

Open-source WhatsApp Web API for building automations, bots, and integrations using modern WebSocket technology.

GitHub Releases Docs

Installation

npm install github:Manuel5906/baileys

Node.js 20+ required.

Quick Start

JavaScript
TypeScript
const { default: makeWASocket, useMultiFileAuthState, DisconnectReason } = require('baileys');

async function start() {
    const { state, saveCreds } = await useMultiFileAuthState('auth_info');
    const sock = makeWASocket({
        auth: state,
        printQRInTerminal: true,
    });

    sock.ev.on('messages.upsert', async ({ messages }) => {
        const m = messages[0];
        if (!m.key.fromMe) {
            await sock.sendMessage(m.key.remoteJid, {
                text: 'Hello from Baileys!'
            }, { quoted: m });
        }
    });

    sock.ev.on('connection.update', ({ connection, lastDisconnect }) => {
        if (connection === 'close') start();
    });
}

start();

Features

📱

Multi-Device

Full WhatsApp multi-device support via WebSocket. No browser required.

🎛️

Interactive Messages

Buttons, lists, native flow, copy-to-clipboard, URL buttons, and more.

🖼️

Media Support

Images, videos, documents, and audio with automatic thumbnail generation.

👥

Groups & Chats

Manage groups, polls, events, albums, stories, and message history.

🔐

Session Management

Persistent auth via multi-file storage, in-memory store, and cache manager.

🔗

Stable Pairing

Robust reconnection logic with customizable pairing for reliable connections.

API Reference

makeWASocket(config)Create a WhatsApp connection with auth and event handlers
sendMessage(jid, content, options)Send any message type (text, image, buttons, etc.)
useMultiFileAuthState(folder)Load/save auth credentials to disk
generateWAMessageFromContent(jid, content, options)Generate a message object without sending
prepareWAMessageMedia(message, options)Upload and prepare media for messages
makeInMemoryStore()In-memory message and chat store
makeCacheableSignalKeyStore(store, logger)Cached wrapper around auth signal key store
