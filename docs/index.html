Baileys

Open-source WhatsApp Web API for building automations, bots, and integrations using modern WebSocket technology.

GitHub Releases Issues

Installation

npm install github:Manuel5906/baileys

Node.js 20+ required.

Quick Start

JavaScript
TypeScript
const { default: makeWASocket, useMultiFileAuthState, DisconnectReason } = require('baileys');

async function start() {
    const { state, saveCreds } = await useMultiFileAuthState('auth_info');
    const sock = makeWASocket({
        auth: state,
        printQRInTerminal: true,
    });

    sock.ev.on('messages.upsert', async ({ messages }) => {
        const m = messages[0];
        if (!m.key.fromMe) {
            await sock.sendMessage(m.key.remoteJid, {
                text: 'Hello from Baileys!'
            }, { quoted: m });
        }
    });

    sock.ev.on('connection.update', ({ connection }) => {
        if (connection === 'close') start();
    });
}

start();

Features

📱

Multi-Device

Full WhatsApp multi-device via WebSocket. No browser needed.

🎛️

Interactive Messages

Buttons, lists, native flow, copy, URL actions and more.

🖼️

Media Support

Images, videos, documents, audio with auto thumbnails.

👥

Groups & Chats

Manage groups, polls, events, albums, stories.

🔐

Session Management

Multi-file auth, in-memory store, cache manager.

🔗

Stable Pairing

Robust reconnection with customizable pairing code.

API Reference

makeWASocket(config)Create a WhatsApp connection
sendMessage(jid, content, options)Send any message type
useMultiFileAuthState(folder)Persistent auth to disk
generateWAMessageFromContent(jid, content, opts)Generate message without sending
prepareWAMessageMedia(msg, opts)Upload and prepare media
makeInMemoryStore()In-memory message store
makeCacheableSignalKeyStore(store, logger)Cached signal key store wrapper
downloadContentFromMessage(msg, type)Download media from message
getContentType(message)Get message content type key
extractUrlFromText(text)Extract URLs from text

Events

connection.updateConnection state changes (open, close, connecting)
messages.upsertNew/updated messages
messages.updateMessage status updates (read, delivered)
creds.updateAuth credentials changed
presence.updateUser presence changes
groups.upsertJoined groups
group-participants.updateGroup participant changes
chats.upsertNew chats
